# My-projected# 📱 سیستەمی بەڕێوەبردنی بازرگانی (Kurdish Business Management App)

ئەمە ئەپڵیکەیشنێکی وێبی سادەیە بۆ بەڕێوەبردنی کارە بازرگانییەکان، کە بە بەکارهێنانی **HTML5, CSS3, و JavaScript** دروستکراوە. گونجاوە بۆ کارکردن لەسەر مۆبایل (Android) لە ڕێگەی Spck Editor.

## ✨ تایبەتمەندییەکان (Features)
* **🛒 بەشی فرۆشتن:** تۆمارکردنی فرۆشتنی ڕۆژانە.
* **💰 بەشی کڕین:** داخڵکردنی کاڵای نوێ بۆ کۆگا.
* **📦 جەردی کۆگا:** بینینی لیستی کاڵاکان و بڕی ماوە.
* **🔄 گەڕانەوە:** بەڕێوەبردنی ئەو مادانەی دەگەڕێنرێنەوە.
* **⚙️ ڕێکخستن:** گۆڕینی ناوی ئەپ و ڕەنگەکان بە ویستی خۆت.
* **📱 دیزاینی Responsive:** بە تەواوی بۆ شاشەی مۆبایل ڕێکخراوە.

---

## 🚀 چۆنیەتی دامەزراندن (Installation)

1.  بەرنامەی **Spck Editor** لە Play Store دابەزێنە.
2.  پڕۆژەیەکی نوێ دروست بکە.
3.  کۆدەکانی ناو فایلی `index.html` کۆپی بکە و پاشەکەوتی بکە.
4.  دوگمەی **Run** داگرە بۆ بینینی ئەپەکە.

---

## 🛠 زمانە بەکارهاتووەکان (Tech Stack)
* **HTML5** - بۆ پێکهاتەی ئەپەکە.
* **CSS3** - بۆ دیزاین و ڕەنگەکان.
* **JavaScript** - بۆ کارکردنی بەشەکان و پاشەکەوتکردنی داتا (LocalStorage).
